# KPMU University College Website

Production-ready Next.js website for KPMU University College (Zambia), built with the App Router and Tailwind CSS.

## Tech Stack

- Next.js
- React
- TypeScript
- Tailwind CSS

## Pages

- Home
- About
- Programmes
- Admissions
- Learning Experience
- Contact

## Local Development

```bash
npm install
npm run dev
```

## Production Build

```bash
npm run build
npm run start
```
