# KPMU University College Website Blueprint

## 1. Institutional Positioning

KPMU University is an AI-powered, future-focused university building practical leaders for Zambia, Africa, and the world.

The university integrates agriculture, business, and computer science with digital systems, real-world application, and modern learning technologies.

---

## 2. Core Objectives of the Website

The website must:

- Establish institutional credibility
- Present KPMU as modern and future-oriented
- Attract and convert prospective students
- Support admissions processes
- Communicate academic programmes clearly
- Present KPMU to investors and partners
- Prepare for integration with:
  - Student portal (portal.kpmuc.com)
  - Learning system (learn.kpmuc.com)

---

## 3. Domain Structure

- www.kpmuc.com → Public website
- portal.kpmuc.com → Student & admin portal (future)
- learn.kpmuc.com → Learning management system (future)

## 7. Design Guidelines

- Clean, modern, premium layout
- Minimalist design (inspired by OpenAI and top universities)
- Strong typography
- Mobile-first responsiveness
- Clear call-to-action buttons
- Use real-world imagery (students, agriculture, computing)
- Avoid clutter

---

## 8. Technology Requirements

- Built using React or Next.js
- Responsive design (mobile, tablet, desktop)
- Component-based structure
- Ready for Azure deployment
- SEO-friendly structure
- Fast loading performance

---

## 9. Integration Readiness

The website must be prepared to integrate with:

- Student portal (applications, accounts)
- Moodle LMS (learning)
- Microsoft 365 (authentication)
- Future payment systems
